Index100

Learning app for Japanese old fashioned Card Game "Hyaku-nin-Isshu".

Download this project as a .zip file Download this project as a tar.gz file

こんなiPhoneアプリを作っています。

百人一首の決まり字を覚えようとしている人のための、覚えた決まり字をテストするアプリです。 ランダムな順番で表示される取り札(下の句)を見て、その歌の決まり字を入力していきます。

設定できること

テストに使う歌の制限

100首全てを覚えていなくても大丈夫。覚えている歌だけを選んでテストすることができます。

今からテストする歌の数

「選んだ歌を全部テストする」こともできますし、「時間がないから今は5首だけテストする」こともできます。

間違えたらすぐ指摘

違う歌の決まり字を入力してしまったとき、すぐに指摘するかどうかを設定できます。 「決まり字はちゃんと覚えてるけど、ちょっと字を押し間違えただけ！」のような時にもいちいち指摘されるのが煩わしい場合は、 offに設定しましょう。「これで決まり！」ボタンを押すまでは、間違いを指摘しません。

効果音

一つ一つの字を選んだときの効果音を選ぶことができます。
